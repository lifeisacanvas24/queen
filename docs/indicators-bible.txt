## **üî• EARLY SIGNAL #1: Volume Dry-Up (VDU) + Pocket Pivot (48-72 Hours Early)**

### **How It Works:**
- **VDU Pattern**: Volume drops to **multi-week lows** while price forms 3-5 tightening contractions (each pullback smaller: e.g., -15%, -8%, -4%, -2%)
- **Signal**: When a **single up-day volume bar exceeds ANY down-day volume** in the prior 10 sessions **while price is still inside the base**
- **Action**:

# COMPLETE BREAKOUT ALGORITHM
class BreakoutScorer:
    def __init__(self, ticker, timeframe):
        self.ticker = ticker
        self.tf = timeframe

    def calculate_score(self):
        score = 0

        # Volume
        if self.volume > self.avg_volume * 3: score += 30
        elif self.volume > self.avg_volume * 2: score += 20
        elif self.volume > self.avg_volume * 1.5: score += 10
        if self.is_vdu_pattern(): score += 15

        # Price/CPR
        if self.cpr_width < self.atr * 0.3: score += 15
        if self.close > self.TC(15m) and self.close > self.TC(1h): score += 15
        if self.is_retest_successful(): score += 20

        # Momentum
        rsi = ta.rsi(self.close, 14)
        if 50 < rsi < 70: score += 20
        if ta.adx(14) > 25: score += 10

        # Futures
        if self.futures_premium > 0.002: score += 10
        if self.open_interest > self.open_interest[1] * 1.05: score += 10

        # Options
        if self.call_put_ratio > 2: score += 10
        if self.iv_rank < 30: score += 5

        # Bonus
        if self.ticker in ["SPY","QQQ"] and self.vix < 20: score += 5

        return score

    def should_trade(self):
        return self.calculate_score() >= 75

    def place_order(self):
        if self.should_trade():
            # Conditional order logic
            entry_price = self.high + self.high * 0.003  # 0.3% above
            stop_price = self.low - self.low * 0.005      # 0.5% below
            quantity = (0.02 * self.capital) / (entry_price - stop_price)

            broker.place_buy_stop(
                symbol=self.ticker,
                price=entry_price,
                stop=stop_price,
                quantity=quantity,
                condition="GTC"
            )
